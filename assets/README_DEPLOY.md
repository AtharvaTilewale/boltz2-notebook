# Boltz2 Notebook Website Package

Files included:
- `index.html` — landing page
- `styles.css` — styling and responsive layout
- `script.js` — mobile navigation and reveal effects
- `assets/boltz2-mark.svg` — simple brand mark

## How to use

### Option 1: local preview
Open `index.html` in a browser.

### Option 2: GitHub Pages / Netlify / Vercel static hosting
Upload the full folder contents and set `index.html` as the entry file.
